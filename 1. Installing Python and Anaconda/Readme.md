# Python Setup Guide with Anaconda & Spyder

This repository provides a beginner-friendly guide to install **Python** using the **Anaconda Distribution**, and run Python code using the **Spyder IDE** (included with Anaconda). Perfect for students, researchers, and developers starting out with Python in a scientific or data analysis environment.

---

## Step 1: Download Anaconda

1. Visit the official Anaconda website:  
   [https://www.anaconda.com/products/distribution](https://www.anaconda.com/products/distribution)
2. Click **Download** and choose your operating system.
3. Select the **64-bit Graphical Installer** for Python 3.x.

---

## Step 2: Install Anaconda

1. Run the downloaded installer.
2. Follow the on-screen instructions:
   - Choose **"Just Me"** during setup (recommended).
   - Keep the default install location.
   - On “Advanced Options”:
     - Leave “Add Anaconda to my PATH” **unchecked**.
     - Keep “Register Anaconda as default Python” **checked**.
3. Click **Install** and wait for completion.

---

## Step 3: Launch Anaconda Navigator

1. Open the **Start Menu** (Windows) and search for **Anaconda Navigator**.
2. Launch it. You’ll see tools like:
   - Jupyter Notebook
   - Spyder
   - VS Code

---

## Step 4: Use Spyder to Run Python Code

1. In Anaconda Navigator, click **Launch** under **Spyder**.
2. In the Spyder IDE:
   - Write Python code in the editor (top left).
   - Run your code using the green "Run" button or press **F5**.
   - View the output in the bottom-right **console**.
   - Use the **Variable Explorer** to inspect variables in real time.
